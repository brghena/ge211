Random Class Reference

Detailed Description

A pseudo-random number generator.

This class has member functions for generating random numbers.

Classes derived from Abstract_game can access an instance of this class via Abstract_game::get_random(), which returns a reference to a Random object maintained by Abstract_game. There is no way for clients to construct their own instances of the Random class.

The right bound is inclusive for integral types but exclusive for floating point types. The left bound is always inclusive.
